 🌍 Language Translation Tool

## Overview

The Language Translation Tool is an AI-powered application that enables users to translate text between multiple languages using translation APIs.

This project was developed as part of my AI Internship program to gain practical experience in API integration, NLP applications, and Python development.

## Features

* Multi-language text translation
* User-friendly interface
* Fast API-based processing
* Real-time translated output
* Easy-to-use workflow

## Technologies Used

* Python
* Translation API
* Tkinter / Flask
* Requests Library

## Project Workflow

1. User enters text.
2. Source and target languages are selected.
3. Request is sent to Translation API.
4. API processes translation.
5. Result is displayed instantly.
